## 具体改动

### 1. `saveUserPreferences()`（第 220-232 行）
将 `currentStatusFilter` 加入 preferences 对象：
```javascript
const preferences = {
    view: currentView,
    sort: currentSort,
    order: currentOrder,
    statusFilter: currentStatusFilter
};
```

### 2. `loadUserPreferences()`（第 235-269 行）
在恢复排序设置后，增加恢复状态筛选逻辑：
```javascript
if (preferences.statusFilter && preferences.statusFilter !== 'all') {
    currentStatusFilter = preferences.statusFilter;
    // 设置对应按钮为 active
    const btn = document.querySelector(`.status-filter-btn[data-status="${preferences.statusFilter}"]`);
    if (btn) {
        document.querySelectorAll('.status-filter-btn').forEach(b => b.classList.remove('active'));
        btn.classList.add('active');
    }
    // 重新执行搜索过滤以应用状态筛选
    const searchInput = document.getElementById('searchInput');
    performSearch(searchInput ? searchInput.value.trim() : '');
}
```

### 3. `initStatusFilterButtons()`（第 306-328 行）
在点击事件中调用 `saveUserPreferences()`：
```javascript
currentStatusFilter = this.dataset.status;
saveUserPreferences();  // 新增
```

### 4. 时序处理
`initFolderStatus()` 是异步加载文件夹状态数据的，而 `loadUserPreferences()` 在它之后同步调用。需要在 `initFolderStatus()` 数据加载完成后，再次检查并应用已保存的状态筛选：
```javascript
// 在 initFolderStatus() 的数据加载成功后添加：
const saved = JSON.parse(localStorage.getItem('folderListPreferences') || '{}');
if (saved.statusFilter && saved.statusFilter !== 'all') {
    currentStatusFilter = saved.statusFilter;
    const btn = document.querySelector(`.status-filter-btn[data-status="${saved.statusFilter}"]`);
    if (btn) {
        document.querySelectorAll('.status-filter-btn').forEach(b => b.classList.remove('active'));
        btn.classList.add('active');
    }
    const searchInput = document.getElementById('searchInput');
    performSearch(searchInput ? searchInput.value.trim() : '');
}
```
